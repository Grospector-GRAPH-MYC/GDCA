# Project Helpers Skill

Common tasks and utilities for the MYC_GDCA backtesting project.

## UI/UX Standards

### Mobile-Friendly Requirements

**All HTML pages in this project MUST be mobile-responsive.** Follow these standards:

#### Required Responsive Breakpoints

```css
/* Tablet */
@media (max-width: 1024px) {
  ...;
}

/* Mobile */
@media (max-width: 768px) {
  ...;
}

/* Small Mobile */
@media (max-width: 480px) {
  ...;
}
```

#### Required Components for Mobile

1. **Hamburger Menu Navigation**

   - Add `.menu-toggle` button that appears on mobile
   - Navigation collapses into dropdown menu on screens < 768px
   - Include toggle JavaScript in base template

2. **Responsive Grids**

   - Strategy cards: Single column on mobile
   - Stats grid: 2 columns max on mobile
   - Charts: Stack vertically, minimum height preserved

3. **Typography Scaling**

   - Hero titles: Desktop 48px → Mobile 26-32px
   - Body text: Scale down by 10-20% on mobile
   - Use `word-break: break-all` for long numbers

4. **Touch-Friendly Targets**
   - Minimum button/link size: 44x44px
   - Adequate spacing between interactive elements

#### Template Structure

All pages extend `templates/base.html` which includes:

- Mobile menu toggle button and JavaScript
- Responsive base styles
- Common header/footer styling

When creating new pages, always:

1. Extend `base.html`
2. Test at 375px width (iPhone SE)
3. Test at 768px width (iPad)
4. Verify hamburger menu works

## Common Tasks

### Task 1: Update Market Data

Download the latest OHLCV data before running backtests:

```bash
poetry run python scripts/download_data.py
```

### Task 2: Quick Backtest Cycle

Run a complete backtest cycle with fresh data:

```bash
poetry run python scripts/download_data.py
poetry run python scripts/run_backtest.py
```

### Task 3: Compare Strategies

After running backtests, open `comparison_result.html` to compare GDCA vs DCA performance.

### Task 4: Mobile Testing

To test mobile responsiveness locally:

1. Open any HTML file in browser
2. Use DevTools (F12) → Toggle device toolbar (Ctrl+Shift+M)
3. Select preset device or set custom width (375px for mobile)
4. Verify hamburger menu and layout adjustments
